# 更新日志

本文件记录 Lite DeepLearning Studio 的重要变更，版本遵循语义化版本号（[SemVer](https://semver.org/lang/zh-CN/)）。

## [0.5.0] - 2026-06-13

补齐教育模块 PRD 的「互动演示」与「实验」部分。至此教育模块 PRD 除视频外已基本完成。

### 新增

- **训练「和上次相比」**：每次训练后，若之前训过，展示准确率前后对比条形图，并自动说明这次改了什么（数据量 / 模型 / 特征）——把「加点数据看看提升」变成可观察的实验（对应实验卡片 E01/E04/E05）。
- **测试集累计抽样**：导入带测试集的数据集后，第 3 步可反复「随机抽一题」，实时累计「答对 / 总数 / 正确率」，亲身体会「单次测试不可靠、多抽才客观」（E07）。
- **词权重可视化**：文本分类用线性模型训练后，展示每个类别「最有代表性的词」，让学生看见模型到底看重哪些词。
- **深度学习地图互动演示**（`static/learn.js`）：①拖动滑块看「数据量 vs 准确率」曲线，理解数据量如何决定上限、何时到瓶颈；②点击神经网络的输入，看信号一层层「前向传播」到输出；③切换逻辑回归 / 决策树 / K 近邻 / 过拟合，看不同算法在同一份数据上画出的分界线。
- **Python 代码片段**：深度学习地图页新增「同样的事，用 Python 怎么写」只读代码，把「点按钮」和「写代码」连起来。

### 备注

- 教育模块 PRD（`docs/PRD_EDUCATION.md`）除「视频」外已全部实现，详见 PRD 顶部「实施进度」。视频因需真实录屏、托管成本高、易随 UI 过时，待平台稳定后再单独立项。

## [0.4.0] - 2026-06-13

把平台从「能训练」扩展到「能教学」：先把通用教育与竞赛分开，再围绕「带学生熟悉深度学习」补齐科普、实验与可解释性内容。

### 变更

- **首页竞赛专区暂时隐藏**：聚焦打磨通用教育模块；竞赛入口待其成熟后在独立页面重新启用（代码保留，仅注释隐藏）。

### 新增

- **深度学习地图页**（`/learn/deep-learning`）：6 节图文科普——传统机器学习 vs 深度学习、神经网络长什么样、图像为什么用 CNN、文本技术的演进、声音怎么变成图片、你在哪里·去哪里。全程内联 SVG 图示，不需写代码；顶栏导航和首页横幅均可进入。
- **每个通用任务的教育内容**：介绍页新增「生活中的例子」；项目页新增可折叠的「动手做几个小实验」（控制变量做对比）、「常见误区」，以及导出步骤的「下一步探索」（引向深度学习地图）。
- **混淆矩阵**：4 个分类任务训练后展示「模型把哪两类搞混了」——绿色对角线为判对、橙色为判错；每类数据充足时用交叉验证预测，更接近真实表现。
- **算法对比结果解读**：「对比所有模型」在成绩单下方多给一句中文解读（谁最稳妥、谁最快、如何按准确率/速度取舍）。
- **新手教程扩充**：第 1 章「传统编程 vs 机器学习」伪代码对照、第 2 章「做个会做实验的人」、术语表新增「欠拟合 / 超参数 / 迁移学习 / 混淆矩阵」、第 6 章「读懂混淆矩阵」图示。
- **教育模块 PRD**：`docs/PRD_EDUCATION.md` 规划了实验卡片、媒体形式四阶段路线图与优先级矩阵。

## [0.3.0] - 2026-06-13

围绕「用真实数据集打磨流程」和「把算法讲清楚」的一次扩展：学生可以直接导入老师整理好的大数据集、用真实测试集检验效果，并在训练页看懂可选的多种算法。

### 新增

- **从整理好的数据集导入**（第 1 步）：在「内置示例」之外新增一条主入口，一键把老师整理好的数据集导入项目。支持每类抽样上限（轻量 100 / 标准 300 / 完整不限）防止大数据集（如 MNIST 约 5.6 万张）训练卡死；加了上限时随机抽取，重新导入可换一批样本做对照实验；采用替换式导入，导入第二个数据集不会和第一个混在一起。
- **测试集抽样检验**（第 3 步）：导入带 `test/` 划分的数据集后，可一键抽一条没参与训练的测试样本，同时显示真实答案与模型预测。测试集存放在 `dataset_eval/`（独立于 `dataset/`），既不参与训练也不会被打进导出包。
- **算法卡片全集**：4 个分类任务的训练页展示完整算法图谱并配中文讲解（作用 / 效能 / 简史 / 优缺点）。经典 CPU 算法可直接训练，新增**梯度提升（GBDT）**、**多层感知机（MLP）**、**支持向量机（SVM）**；深度模型（MobileNet、ResNet、ViT、CLIP、BERT、LSTM 等）以置灰卡片展示，让学生看到完整技术版图（暂不支持在平台内训练）。
- **图像特征提取器可选**：图像任务可在「MobileNet 迁移学习」与「像素特征」之间选择，方便对照讲解「为什么要迁移学习」。
- **可迁移部署**：新增 `LDS_DATASETS_ROOT` 环境变量指定数据集根目录（默认项目根的 `datasets/`，已加入 .gitignore），docker-compose 支持挂载；路径不存在时导入入口自动隐藏。
- **文档**：新增 `docs/DATASET_FORMAT.md`（整理数据集的标准格式规范）与 `docs/ALGORITHM_CATALOG.md`（算法全集说明）。

### 改进

- **对比所有模型提速**：先分层抽样到约 200 条再赛跑，无论数据集多大都不再卡顿；图像/音频在读取文件字节之前就抽样，避免把几百 MB 数据全部读入内存。

### 修复

- **传感器 CSV 超长保存失败**：CSV 长度上限从 2 万字符放宽到 200 万字符，2000 行投掷数据等大表不再返回 422；相关错误也不再显示为 `[object Object]`。
- **交叉验证跳过提示纠错**：提示文案的阈值更正为「每类至少 3 条」（原误写为 10 条），措辞也更柔和。

## [0.2.0] - 2026-06-12

围绕「教学体验」和「导出流程」的一次集中改进，让学生在不依赖教师口头讲解的情况下也能独立走完一个 AI 项目。

### 新增

- **全局「新手教程」侧栏**：顶栏任意页面可点开右侧抽屉，内含一份理论与实践结合的完整入门教程，分 6 章——什么是机器学习、四步流程详解、五种任务怎么选、核心概念速查（术语表）、常见报错怎么办、怎样算成功与如何改进。
- **通用学习教学内容**：5 个通用任务的介绍页新增「了解原理」概念卡，4 个步骤各新增「了解这一步」可折叠讲解，把每一步对应到机器学习流程。
- **导出包文件说明**：每个材料包内新增 `包内文件说明.md`，按重要程度分 5 组逐文件用学生中文解释作用，并指引「只是练习关注核心运行、上板子加看部署、交比赛再看提交清单」。

### 改进

- **四步工作流强制顺序**：完成前一步后，后续步骤才解锁，未解锁的步骤按钮置灰不可点，避免乱序操作。
- **导出 / 下载按钮**：改为两个始终固定显示的按钮，导出前下载按钮置灰禁用、导出成功后激活；去掉了原先动态弹出下载链接、两个按钮挤在一起的设计。
- **导出包命名**：压缩包文件名从不透明的项目 ID 改为「项目名_时间戳」（如 `我的智能问答作业_20260612-195520.zip`），便于区分不同批次的下载；每次导出生成新批次而非覆盖。

### 修复

- **并发保存导致数据损坏**：修复编辑器重建（如加载样本包后）重复绑定保存监听器，导致一次点击触发多个并发保存、相互竞争把数据文件写坏的问题（表现为「操作失败，请重试」）。所有数据集、元数据、CSV、标签文件的写入改为原子写（临时文件 + `os.replace`），杜绝并发写入或读到半写文件造成的损坏。

## [0.1.0]

- 初始版本：四步在浏览器内完成 AI 项目（准备数据 → 训练模型 → 测试效果 → 导出材料）；支持文本/图像/语音分类、问答检索、传感器决策、OCR 查错等能力的在线训练；导出可运行的 Python 材料包与行空板 M10 部署层；智能博物、优创未来两个竞赛版本单代码库切换。
