# 更新日志

本文件记录 Lite DeepLearning Studio 的重要变更，版本遵循语义化版本号（[SemVer](https://semver.org/lang/zh-CN/)）。

## [0.2.0] - 2026-06-12

围绕「教学体验」和「导出流程」的一次集中改进，让学生在不依赖教师口头讲解的情况下也能独立走完一个 AI 项目。

### 新增

- **全局「新手教程」侧栏**：顶栏任意页面可点开右侧抽屉，内含一份理论与实践结合的完整入门教程，分 6 章——什么是机器学习、四步流程详解、五种任务怎么选、核心概念速查（术语表）、常见报错怎么办、怎样算成功与如何改进。
- **通用学习教学内容**：5 个通用任务的介绍页新增「了解原理」概念卡，4 个步骤各新增「了解这一步」可折叠讲解，把每一步对应到机器学习流程。
- **导出包文件说明**：每个材料包内新增 `包内文件说明.md`，按重要程度分 5 组逐文件用学生中文解释作用，并指引「只是练习关注核心运行、上板子加看部署、交比赛再看提交清单」。

### 改进

- **四步工作流强制顺序**：完成前一步后，后续步骤才解锁，未解锁的步骤按钮置灰不可点，避免乱序操作。
- **导出 / 下载按钮**：改为两个始终固定显示的按钮，导出前下载按钮置灰禁用、导出成功后激活；去掉了原先动态弹出下载链接、两个按钮挤在一起的设计。
- **导出包命名**：压缩包文件名从不透明的项目 ID 改为「项目名_时间戳」（如 `我的智能问答作业_20260612-195520.zip`），便于区分不同批次的下载；每次导出生成新批次而非覆盖。

### 修复

- **并发保存导致数据损坏**：修复编辑器重建（如加载样本包后）重复绑定保存监听器，导致一次点击触发多个并发保存、相互竞争把数据文件写坏的问题（表现为「操作失败，请重试」）。所有数据集、元数据、CSV、标签文件的写入改为原子写（临时文件 + `os.replace`），杜绝并发写入或读到半写文件造成的损坏。

## [0.1.0]

- 初始版本：四步在浏览器内完成 AI 项目（准备数据 → 训练模型 → 测试效果 → 导出材料）；支持文本/图像/语音分类、问答检索、传感器决策、OCR 查错等能力的在线训练；导出可运行的 Python 材料包与行空板 M10 部署层；智能博物、优创未来两个竞赛版本单代码库切换。
