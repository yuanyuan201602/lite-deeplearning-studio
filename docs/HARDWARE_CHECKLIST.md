# 行空板 M10 + DFRobot 外设硬件选择清单

本文用于给老师和学生选择智能博物、优创未来两套轻量版所需硬件。

统一硬件基线：

- 主控：行空板 M10
- 外设：DFRobot 开源硬件外设，优先选择 Gravity 系列模块
- 软件：智能博物轻量版或优创未来轻量版

采购前请以 DFRobot 官方页面的库存、版本和接口说明为准。

## 2. 基础必备清单

每个学生小组建议至少准备：

| 硬件 | 数量 | 用途 | 备注 |
|---|---:|---|---|
| 行空板 M10 | 1 | 主控、屏幕显示、运行 Python 项目 | 两个比赛都必备 |
| USB 数据线 | 1 | 供电、连接电脑、拷贝文件 | 建议质量稳定 |
| Windows 学生电脑 | 1 | 安装软件、生成任务包、训练模型 | 推荐 Windows 10/11 |

## 3. 智能博物推荐组合

智能博物重点是“识别 + 显示 + 播报”。

### 3.1 标准组合

| 硬件 | 数量 | 对应任务 |
|---|---:|---|
| 行空板 M10 | 1 | 所有挑战 |
| DFRobot USB 摄像头或兼容摄像头 | 1 | 挑战一、二、三、四 |
| DFRobot Gravity 语音合成模块 V2.0 | 1 | 全部播报 |
| DFRobot Gravity I2C 语音录放模块 | 1 | 可替代或补充语音输出 |
| 非遗人物卡片 | 5 到 10 张 | 挑战一 |
| 非遗知识卡片 | 5 到 10 张 | 挑战二 |
| 非遗词语卡片 | 6 到 20 张 | 挑战三 |
| 非遗句子卡片 | 6 到 20 张 | 挑战四 |

### 3.2 任务对应关系

| 智能博物任务 | 必备外设 | 可选外设 |
|---|---|---|
| 挑战一：认识非遗传承匠人 | 摄像头、语音合成模块 | 补光灯、按钮 |
| 挑战二：了解非遗专业知识 | 摄像头、语音合成模块 | 补光灯 |
| 挑战三：非遗文化分类学览 | 摄像头或文本输入、语音合成模块 | 按钮 |
| 挑战四：非遗文化深化认知 | 摄像头或文本输入、语音合成模块 | 按钮 |
| 创意拓展：非遗问答 | 语音合成模块 | 离线语音识别模块、按钮 |

### 3.3 采购建议

优先买：

1. 行空板 M10
2. DFRobot USB 摄像头
3. DFRobot Gravity 语音合成模块 V2.0
4. Gravity 连接线

如果预算充足：

- 加一个离线语音识别模块，用于创意拓展问答。
- 加一个按钮模块，用于“开始挑战”“下一张卡片”。
- 加一个补光灯或 LED 模块，提高卡片拍摄稳定性。

## 6. 传感器和执行器建议

### 6.1 传感器

可选：

- 温度传感器：用于智慧医疗体温或环境判断。
- 距离传感器：用于判断物体位置或机械安全距离。
- 光线传感器：用于环境状态判断。
- 按钮模块：用于开始、确认、重置。
- 触摸传感器：用于交互输入。

### 6.2 执行器

可选：

- LED 灯模块：状态提示。
- 蜂鸣器：声音提示。
- 舵机：夹取、转动、指示。
- DC 电机：移动或传送。
- 震动马达：触觉反馈。

### 7.1 入门包

适合先完成软件流程和简单展示：

- 行空板 M10
- 摄像头
- DFRobot 语音合成模块
- LED 或按钮模块

### 7.2 标准比赛包

适合覆盖两个比赛的大多数任务：

- 行空板 M10
- 摄像头
- DFRobot 语音合成模块
- DFRobot 离线语音识别模块
- Gravity 传感器套件
- 按钮、LED、蜂鸣器

### 7.3 高中拓展包

适合高中组运动控制和机械结构：

- 标准比赛包
- 行空板 M10 扩展板或电机驱动扩展
- 舵机或 DC 电机
- 距离传感器
- 机械结构件
- 安全停止按钮

## 8. 采购前确认清单

下单前请确认：

- 模块是否支持行空板 M10 或可通过 USB/I2C/UART/GPIO 接入。
- 模块是否有 Python 示例或 Arduino 示例可参考。
- 是否需要额外供电。
- 是否需要 Gravity 转接线。
- 是否需要扩展板。
- 语音合成模块是否支持中文。
- 摄像头是否能被行空板 M10 识别。
- 电机和舵机是否有足够电流。
- 是否准备了备用线材和备用电源。

## 9. 课堂备件建议

建议老师额外准备：

- USB 数据线 2 到 3 根
- Gravity 连接线若干
- 备用摄像头 1 个
- 备用语音模块 1 个
- 备用按钮和 LED 模块
- 移动电源
- 胶带、扎带、螺丝刀、热熔胶等固定工具

## 11. 注意事项

- 本清单是教学和比赛准备建议，不等于必须全部购买。
- 同一功能可以用不同 DFRobot 模块替代。
- 高中组机械夹取对供电和结构稳定性要求更高，建议提前单独调试。
- OCR、图像识别和大模型互动都应先在电脑端跑通，再迁移到行空板 M10。
