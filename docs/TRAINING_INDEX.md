# 培训文档索引

本目录提供两套面向学生的完整培训文档。

## 智能博物

使用安装包：

```text
SmartMuseum-Windows-Setup.zip
```

培训文档：

[智能博物轻量版学生培训文档](TRAINING_SMART_MUSEUM.md)

适用任务：

- 挑战一：认识非遗传承匠人
- 挑战二：了解非遗专业知识
- 挑战三：非遗文化分类学览
- 挑战四：非遗文化深化认知
- 创意拓展：非遗知识问答

## 优创未来

使用安装包：

```text
FutureCreator-Windows-Setup.zip
```

培训文档：

[优创未来轻量版学生培训文档](TRAINING_FUTURE_CREATOR.md)

适用任务：

- 小学组：语音互动与单类图像识别
- 初中组：视觉模型训练与调用
- 高中组：大模型语音互动、视觉识别与运动控制
- 图像识别快速模板
- 传感器决策程序模板

## 统一硬件基线

两套培训都默认使用：

- 行空板 M10
- DFRobot 开源硬件外设

## 学生安装入口

Windows 学生机解压对应 zip 后：

1. 双击 `一键安装.bat`
2. 双击 `启动软件.bat`
3. 打开 `http://127.0.0.1:8000`

如果需要 OCR 错字识别，双击：

```text
安装OCR增强.bat
```
